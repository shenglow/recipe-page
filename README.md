### What I learned

In this challenge, I learned more about the usage of `dd` element. For example, when presenting information in a simple table without headers, using the `dd` element is more appropriate.

```html
<dl>
  <dt>Calories</dt>
  <dd>277kcal</dd>
</dl>
<dl>
  <dt>Carbs</dt>
  <dd>0g</dd>
</dl>
<dl>
  <dt>Protein</dt>
  <dd>20g</dd>
</dl>
```
